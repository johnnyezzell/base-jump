Welcome to Base Jump.

Base Jump is a Java Script library that is designed to provide the functions and data types JavaScript are missing. Base Jump is still in active development and not ready for use in production.

Base Jump will provide the following:

  • Common Data Structures (BST, Linked List, etc.)
  • Lots of utilities through prototype extensions (Let the trolls fly!)
  • 100% Code Coverage
  • Clean Documentation
